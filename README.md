## üåü Introduction
FinBERT-Long-Text is a Python tool that helps you analyze long news articles related to finance. Unlike the standard BERT model, which struggles with lengthy texts, this wrapper handles news articles that exceed the regular 512-token limit. It focuses on critical news without losing valuable insights to filler text. This can greatly aid in investment analysis, sentiment detection, and more.
